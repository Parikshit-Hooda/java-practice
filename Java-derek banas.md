# Notes/Points
- Don't use `==` to compare two String variables. Use `.equals`. For example:
  - `System.out.println("dog equals cat:" + "dog".equals("cat"));`
  - To ignore case, `System.out.println("dog equals cat:" + "dog".equalsIgnoreCase("cat"));`
